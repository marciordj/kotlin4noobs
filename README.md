## Sobre o Projeto
Kotlin é uma linguagem relativamente nova, criada em 2011 pela JetBrains e que foi adotada pelo Google como a linguagem oficial para desenvolvimento nativo Android. Mas ela não é usada somente no desenvolvimento Android, atualmente há vários projetos que adotam kotlin no back-end, ou até mesmo web. Aqui vou tentar dar uma base de como funciona a linguagem mais voltada para o Desenvolvimento Android, porém pode ser aplicada para outras áreas também.
